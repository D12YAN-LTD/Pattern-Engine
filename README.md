# Pattern Engine

Pattern Engine is a TradingView Pine Script project that detects
institutional momentum patterns inspired by Qullamaggie and similar
high-momentum growth strategies.

The long-term objective is to build a production-quality screener
capable of identifying stocks progressing through defined checkpoints.
